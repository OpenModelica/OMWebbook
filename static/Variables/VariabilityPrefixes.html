Variability Prefixes


The prefixes constant, parameter, discrete of a variable declaration are called variability prefixes and define in which situations the value of a variable may change and when it is kept constant. A constant has the least variability. A parameter has some more variability since it can be changed between simulations. Discrete variables are variables that change value only at event instants. Boolean, Integer, and String variables are discrete-time by default.
