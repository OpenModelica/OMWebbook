Flexible Array Sizes


The possibility to declare arrays with unspecified dimension sizes is quite important in order to design flexible models that can adapt to different problem sizes. An unspecified dimension size is stated by using a colon (:) instead of the usual integer expression for dimension size. The unspecified dimension sizes will eventually be determined when the array variable is bound to some value.
