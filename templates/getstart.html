A cell containing a Modelica model, class, or function has to be evaluated before it is possible to simulate it. To do this, click inside the cell and push shift-enter or click evaluate button. Then you have to evaluate a simulate command, e.g. by typing "simulate(modelname, startTime=0, stopTime=25 )", After simulating a class it is possible to plot or just look at the values of the variables in the class by using the plot command. The plot is generated with the help Dygraph, a javascript plugin library used for generating graphs in web browsers.
The functionalities Add cells, Delete Cells , Insert Cells between and save to pdf are available only for DynamnicWebbook.
